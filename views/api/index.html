How to use the Shrnk API & microservice

# Microservice

Shorten

1. Make a GET request 

curl https://shrnk.glitch.me/shrink/[URL]

The server will respond with a JSON object containing the original and shortened URL.

Redirect

1. Make a GET request 

curl https://shrnk.glitch.me/[encodedUrl]

The server will respond with a redirect to the original URL.

# API

1. Make a POST request 

curl -d "url=http://www.google.com" https://shrnk.glitch.me/api/shrink

The server will respond with a JSON object containing the original and shortened URL.
